## 🚀 Overview

This repository contains hands-on implementations of fundamental Machine Learning concepts, including supervised and unsupervised learning techniques. It also includes mini-projects and practical notebooks to understand real-world applications.

## ▶️ How to Run

1. Clone the repository

git clone https://github.com/Sidtripathi07/Machine_learning_using_python.git


2. Navigate to the project folder

cd Machine_learning_using_python


3. Install dependencies

pip install numpy pandas matplotlib seaborn scikit-learn


4. Run Jupyter Notebook

jupyter notebook
